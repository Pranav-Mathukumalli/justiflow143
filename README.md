# JustiFlow — AI Fairness Auditing Platform

## Overview
[cite_start]JustiFlow is a Flask-based platform designed to detect and mitigate algorithmic bias in datasets, aligning with the EU AI Act.

## Features
* [cite_start]**DIR, SPD, and EOD Metrics**: Canonical fairness measurement.
* [cite_start]**Mitigation**: Reweighing, Oversampling, and Threshold Adjustment.
* [cite_start]**AI Audits**: Plain-English reports via Claude API.

## Setup
1. [cite_start]Install dependencies: `pip install -r requirements.txt` 
2. [cite_start]Run the application: `python app.py` 
3. [cite_start]Visit `http://localhost:5000` in your browser.
